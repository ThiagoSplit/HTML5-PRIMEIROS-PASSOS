Principais Formatações

NEGRITO / DESTAQUE

Neste texto ou nesta frase tem um pedaço dela em negrito, ou seja eu usei a tag <b> (não semântica).

Já nessa frase tem um termo em destaque usando a tag <strong> (semântica).

OU SEJA A PARA USAR O NEGRITO BASTA EU UTILIZAR QUALQUER UMA DAS TAGS: <b> ou <strong>

ITÁLICO / ÊNFASE

Nesta frase temos um termo em itálico usando a tag <i> (não semântico)

Nesta frase termo em ênfase usando a tag <em> (semântico)

OU SEJA A PARA USAR O ITÁLICO BASTA EU UTILIZAR QUALQUER UMA DAS TAGS: <i> ou <em>

PORÉM RECOMENDA-SE QUE USE <em> PARA ITÁLICO E <strong> PARA NEGRITO, PORQUE COMO O HTML5 É SEMÂNTICO, É BEM PROVÁVEL QUE EM ALGUM MOMENTO AS TAGS <b> E <i> PODEM SE TORNAR OBSOLETAS.
