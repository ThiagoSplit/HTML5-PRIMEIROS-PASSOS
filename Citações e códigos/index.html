Então retornando ao código em Python...

Neste exemplo acima foi utilizado além da tag <code> foi utulizado outra tag a tag <pre>, que serviu para deixar o código do jeito que ele tem que ser escrito.

Caso eu retire a tag <pre> o codigo irá ficar desconfigurado. ficará da seguinte forma:

num = int(input('Escreva a sua número: ')) if int % 2 == 0: print(f'O número {num} é um número PAR!') else: print(f'O número {num}, é um número IMPAR!') print(Acabou aqui o código)
