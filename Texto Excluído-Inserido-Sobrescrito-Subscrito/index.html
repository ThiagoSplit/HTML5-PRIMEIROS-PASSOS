Texto Inserido e Deletado

Texto Deletado

Podemos marcar um texto como excluido usando a tag <del>, indicando que o mesmo deve ser lido mas não considerado

Texto Inserido

Podemos marcar um texto como inserido usando a tag <ins>, indicando que o mesmo foi inserido.

Existe também a tag <u>, que serve para sublinhar. Porém ela é não semântica.

Texto Sobrescrito

Serve para usar as Potências por exemplo: x10+5+9-1. Neste caso foi utilizado a Tag <sup>

Texto Subscrito

Serve para usar as em fórmulas químicas: H20. Neste caso foi utilizado a Tag <sub>
